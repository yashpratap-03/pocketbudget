# PocketBudget

PocketBudget is a simple expense tracker web application built for the IU course **DLBCSPJWD01 (Project Java & Web Development)**.

## Features (MVP)
- Add an expense (amount, category, date, note)
- View expense list
- Monthly total and totals by category

## Tech Stack
- Frontend: HTML, CSS, JavaScript (vanilla)
- Backend: Node.js, Express
- Storage: JSON file (upgradeable to SQLite)

## API Endpoints
- GET /health
- POST /expenses
- GET /expenses
- DELETE /expenses/:id (optional)
